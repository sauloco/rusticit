# 4. DESIGN SYSTEM

Palette:
- Charcoal: `#353535`
- Purple Dark: `#581B98`
- Purple Bright: `#9C1DE7`
- Crimson Accent: `#FF073A`
- Signal Yellow: `#FFF707`
- Offwhite: `#F7F7F7`

Usage guidance:
- Charcoal = base background / primary surfaces
- Offwhite = text on dark sections / clean light contrast
- Purple Dark/Bright = structure + brand accents
- Crimson = strong CTA emphasis / highlights
- Yellow = sparing accent only

Typography:
- Headings: Inter or Rubik
- Body: Inter
- Technical microcopy / tags / metadata: JetBrains Mono or similar

Style rules:
- tight, modern heading tracking
- large but not exaggerated hero scale
- no decorative serif drama
- sharp UI rhythm
- rounded corners in a restrained system
- subtle texture/noise only if it improves depth, never as a gimmick
